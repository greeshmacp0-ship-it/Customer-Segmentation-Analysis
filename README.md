# Customer Segmentation Analysis

## Objective
Perform customer segmentation using clustering techniques to identify distinct customer groups for targeted marketing and retention strategies.

## Tools Used
Orange Data Mining, Python (K-means Clustering), Data Preprocessing

## Dataset
Customer dataset containing demographic and behavioral attributes (3900+ records).

## Steps Performed
- Data cleaning and preprocessing
- Feature selection
- Applied K-means clustering algorithm
- Identified customer segments
- Analyzed demographic and behavioral characteristics

## Key Outcomes
- Identified 3 distinct customer segments
- Improved understanding of customer behavior patterns
- Provided insights for targeted marketing strategies
